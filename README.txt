Django Task Manager is an interactive website that allows you to manage your tasks in an organized and efficient way. It is built using the Django framework and served by Python3 on the backend. The application uses databases to store information and utilises bootstrap, and CSS for a sleek and user-friendly interface.

Getting Started
To start using Django Task Manager, simply follow these steps:

1. Login or create an account by providing your email address and a password.
2. You will receive a welcome email after successful account creation.
3. Once logged in, you can create, view, and manage your tasks.

Features
-Secure authentication system using email and password.
-Welcome email upon successful account creation.
-Ability to create, view, and manage tasks.
-User-friendly interface utilizing Bootstrap and CSS.

Technologies Used
-Django framework
-Python3
-Databases
-Bootstrap
-CSS
